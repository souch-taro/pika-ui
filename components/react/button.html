# API

# PkButton Props

参数 类型 默认值 必填 说明
appParameter string false 打开 APP 时，向 APP 传递的参数。生效时机：open-type='launchApp'
disabled boolean false false 是否禁用
formType 'submit'|'reset' false 用于
组件，点击分别会触发
组件的 submit/reset 事件
hoverClass string button-hover false 指定按下去的样式类。当 hover-class='none' 时，没有点击态效果
hoverStartTime number 20 false 按住后多久出现点击态，单位毫秒
hoverStayTime number 70 false 手指松开后点击态保留时间，单位毫秒
hoverStopPropagation boolean false false 指定是否阻止本节点的祖先节点出现点击态
hoverStyle string none false 由于 RN 不支持 Class，故 RN 端的 Button 组件实现了 hoverStyle 属性，写法和 style 类似，只不过 hoverStyle 的样式是指定按下去的样式
icon React.ReactNode false 图标
lang 'en'|'zh_CN'|'zh_TW' false 指定返回用户信息的语言，zh_CN 简体中文，zh_TW 繁体中文，en 英文。生效时机: open-type='getUserInfo'
loading boolean false false 名称前是否带 loading 图标
openType 'contact'|'contactShare'|'share'|'getRealnameAuthInfo'|'getAuthorize'|'getPhoneNumber'|'getUserInfo'|'lifestyle'|'launchApp'|'openSetting'|'feedback' false 微信开放能力（详见 taro 文档 Button 组件）
plain boolean false false 按钮是否镂空，背景色透明
scope 'userInfo'|'phoneNumber' false 支付宝小程序 scope。生效时机：open-type='getAuthorize'
sendMessageImg string 截图 false 会话内消息卡片图片。生效时机：open-type='contact'
sendMessagePath string 当前标题 false 会话内消息卡片点击跳转小程序路径。生效时机：open-type='contact'
sendMessageTitle string 当前标题 false 会话内消息卡片标题。生效时机：open-type='contact'
sessionFrom string false 会话来源。生效时机：open-type='contact'
showMessageCard boolean false false 显示会话内消息卡片。生效时机：open-type='contact'
size 'small'|'medium'|'large' medium false 按钮的大小
type 'primary'|'secondary'|'linear'|'text' 'primary' false 按钮的样式类型
onContact BaseEventOrigFunction<onContactEventDetail> false 客服消息回调。生效时机：open-type='contact'
onError BaseEventOrigFunction<any> false 当使用开放能力时，发生错误的回调。生效时机：open-type='launchApp'
onGetAuthorize BaseEventOrigFunction<any> false 支付宝获取会员基础信息授权回调。生效时机：open-type='getAuthorize'
onGetPhoneNumber BaseEventOrigFunction<onGetPhoneNumberEventDetail> false 获取用户手机号回调。生效时机：open-type='getphonenumber'
onGetRealnameAuthInfo BaseEventOrigFunction<any> false 获取用户实名。生效时机：open-type='getRealnameAuthInfo'
onGetUserInfo BaseEventOrigFunction<onGetUserInfoEventDetail> false 用户点击该按钮时，会返回获取到的用户信息，回调的 detail 数据与 Taro.getUserInfo 返回的一致。生效时机: open-type='getUserInfo'
onLaunchapp BaseEventOrigFunction<any> false 打开 APP 成功的回调。生效时机：open-type='launchApp'
onOpenSetting BaseEventOrigFunction<onOpenSettingEventDetail> false 在打开授权设置页后回调。生效时机：open-type='openSetting'

# formType

参数 说明
submit 提交表单
reset 重置表单

# lang

参数 说明
en 英文
zh_CN 简体中文
zh_TW 繁体中文

# openType

参数 说明
contact 打开客服会话，如果用户在会话中点击消息卡片后返回小程序，可以从 bindcontact 回调中获得具体信息
contactShare
feedback 打开“意见反馈”页面，用户可提交反馈内容并上传日志，开发者可以登录小程序管理后台后进入左侧菜单“客服反馈”页面获取到反馈内容
getAuthorize
getPhoneNumber 获取用户手机号，可以从 bindgetphonenumber 回调中获取到用户信息
getRealnameAuthInfo
getUserInfo 获取用户信息，可以从 bindgetuserinfo 回调中获取到用户信息
launchApp 打开 APP，可以通过 app-parameter 属性设定向 APP 传的参数
lifestyle
openSetting 开授权设置页
share 触发用户转发，使用前建议先阅读使用指引

# size

参数 说明
small 小
medium 中
large 大

# type

参数 说明
primary 主要
secondary 次要
linear 线性
text 文本

# onContactEventDetail

参数 类型 说明
errMsg string
path string 小程序消息指定的路径
query Record<string, any> 小程序消息指定的查询参数

# onGetPhoneNumberEventDetail

参数 类型 说明
errMsg string
encryptedData string 包括敏感数据在内的完整用户信息的加密数据
iv Record<string, any> 加密算法的初始向量

# onGetUserInfoEventDetail

参数 类型 说明
userInfo { nickName: string; avatarUrl: string; gender: Gender; province: string; city: string; country: string; }
rawData string 不包括敏感信息的原始数据 JSON 字符串，用于计算签名
signature string 使用 sha1(rawData + sessionkey) 得到字符串，用于校验用户信息
encryptedData string 包括敏感数据在内的完整用户信息的加密数据
iv string 加密算法的初始向量
errMsg string

# gender

参数 说明
0 未知
1 男
2 女

# onOpenSettingEventDetail

参数 类型 说明
errMsg string
authSetting Record<string, boolean>
