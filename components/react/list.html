# 说明

# 组件介绍

用于常规列表展示

# 开发人员

章政

# 引入

# 组件

如果需要按需引入, 则:

import { PkList } from '@souche-taro/pika-react/lib';
const { PkListItem } = PkList;

组件依赖的样式文件（仅按需引用时需要）

@import '~@souche-taro/pika-react/dist/style/components/list.scss';
@import '~@souche-taro/pika-react/dist/style/components/icon/iconfont.scss';

# API

# PkList Props

参数 说明 类型 可选值 默认值
hasBorder 是否有边框 boolean true/false true

# PkListItem Props

参数 说明 类型 可选值 默认值
hasBorder 是否有边框 boolean true/false true
disabled 是否禁用 boolean true/false false
note 元素的描述信息 string ''
title 元素的标题 string ''
thumb 元素的主要缩略图 string ''
isSwitch 额外信息是否开关 boolean false
extraText 额外信息的文本 boolean false
extraThumb 额外信息的缩略图 string ''
switchIsCheck 额外信息开关的值 boolean false
switchColor 开关的颜色 string ''
iconInfo icon 信息 见PkIcon
arrow 箭头的方向 string 'up','down','right' 'right'

# PkNavBar Events

事件 说明 返回参数
onClick item点击 event
onSwitchChange switch切换 event
